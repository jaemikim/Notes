1. 소개
  1) KNN 모델
    - K Nearest Neighbors
    - 종속변수가 범주형임
    - 종속변수가 개수가 3개 이상인 다중분류(Multiclassification)
  2) K-최근접 이웃
    - 거리 기반 모델
    - 각 데이터 간의 거리를 활용해서 새로운 데이터를 예측하는 모델
    - 가까이에 있는 데이터를 고려하여 새로운 데이터를 예측하는 모델
      ㆍ이때 가까이에 있는 데이터를 고려하여 예측값이 결정됨
      ㆍK개의 가장 가까운 이웃 데이터에 의해 예측된다는 의미임
    - 데이터 간의 거리를 활용해서 새로운 데이터가 어디에 속하는지 구분하는 알고리즘 

2. ML 순서
  1) 문제 정의
  2) 라이브러리 및 데이터 불러오기
  3) 데이터 확인하기
  4) 전처리
    - 결측지 처리
  5) 스케일링
  6) 데이터 나누기
    - 학습셋과 실험셋(7:3, 8:2)
  7) 데이터 모델링
  8) 모델을 활용해 예측하기
  9) 예측 모델 평가하기
