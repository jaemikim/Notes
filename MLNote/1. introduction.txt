1. 인공지능 > 머신러닝 >ㅓ 딥러닝
  1) 인공지능이 가장 큰 개념임
    - 인간의 뇌를 모방하는 프로그랭 
    - 자율 주행, 알파고, 쇼핑몰 추천 알고리즘 등이 모두 인공지능에 해당
  2) 머신러닝(Machine Learning, ML)
    - 기계학습

    - 비교
      ㆍ전통적인 프로그램
          데이터             -------->
                                       컴퓨터  ------> 결과
          인간이 작성한 모델  -------->
      ㆍ머신 러닝
          샘플 데이터        -------->
                                       컴퓨터  ------> 모델
          정답값            -------->      
      ㆍ예측 
          새로운 데이터      -------->
                                       컴퓨터  ------> 결과
          모델              -------->     

    - 입력값과 해당 정답값에 대한 입력을 해주면, 입력값과 정답값 간의 관계를 찾아서 새로운 프로그램(모델)을 만듦
    - 그 후 새로운 프로그램(머신러닝 모델)에 데이터를 입력해서 값을 예측함
    - 새로운 데이터가 주어졌을 때 결과를 예측하는 일련의 과정임
    
  3) 딥러닝 (Deep Learning)
    - 심층 학습
    - 인공 신경망을 기반으로 한 특수한 머신 러닝 기법
      ㆍ인간의 두뇌 작동 방식을 본떠 개발된 것
    - 빅데이터 기술과 하드웨어 발전, 새로운 알고리즘 등장으로 각광받게 되었음
    - 자연어 처리나 이미지, 비디오 분석 같은 목적으로 사용됨
