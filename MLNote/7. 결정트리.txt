1. 개요
  1) 트리 모델 계열의 근간인 결정 트리
    - 기본 모델(based model)
  2) 결정 트리(Decision Tree)
    - 관측값과 목표값을 연결시켜주는 예측 모델
    - 나무 모양으로 데이터를 분류함
    - 각 변수의 특정 지점을 기준으로 데이터를 분류해가며 예측 모델을 만듦

2. 학습 순서
  1) 문제 정의
  2) 라이브러리 및 데이터 불러오기 & 데이터 확인하기
  3) 전처리
  4) 모델링 및 예측하기

3. 문제 정의 
  1) 현재 내 연봉 수준 적절한가?
    - 나이, 교육수준, 혼인 상태, 직업, 인종 성별 등
  2) 결정 트리 알고리즘 활용해서 연봉 등급을 예측하기

4. 라이브러리 및 데이터 불러오기 & 데이터 확인하기 
  1) 컬럼
    - age : 연령
    - workclass : 고용형태
    - education : 학력
    - education-num : 교육 연수
    - marital-status : 혼인상태
    - occupation : 직업
    - relationship : 가족 관계
    - race : 인종
    - sex : 성별
    - capital-gain : 자산 증가 
    - capital-loss : 자산 감소
    - hours-per-week : 주당 노동 시간
    - native-country : 본국
    - class : 연봉
